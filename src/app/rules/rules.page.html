Regras!

OBJETIVO DO JOGO

Uma Palavra é um party game cooperativo. Como um time, todos jogam para alcançar a melhor pontuação!

Vocês escreverão pistas e ajudarão um dos jogadores – o jogador ativo – a adivinhar uma palavra Secreta.

Escolham as pistas em segredo, sem interagir com os outros do time! Sejam originais para que a mesma palavra não seja escrita por outra pessoa. Pistas idênticas serão anuladas antes de o jogador ativo poder vê-las.

No final do jogo, registre sua pontuação de acordo com o número de palavras Secretas descobertas.

VISÃO GERAL

  • 1) O jogador ativo passa o dispositivo para os companheiros verem a palavra secreta.
  • 2) Cada jogador escreve uma pista sem mostrá-la para ninguém e sem interação entre o time. A pista deve ser composta de uma única palavra. Observação: um dígito, um número, uma sigla, uma onomatopeia ou um caractere especial são todos considerados palavras. Exemplo: A pista 007 é permitida para a palavra Agente, as pistas Trimmmm ou SMS são permitidas para a palavra Telefone, e a pista $ é permitida para a palavra Dinheiro.
  • 2.1) Pistas inválidas: A palavra Secreta escrita de um jeito diferente ( a pista Quarta não pode ser usada se a palavra a ser adivinhada for Quarto), A palavra Secreta escrita em outro idioma (a pista House não pode ser usada se a palavra a ser adivinhada for Casa), Uma palavra da mesma família da palavra Secreta ( a pista Princesa não é permitida se a palavra a ser adivinhada for Príncipe), Uma palavra inventada ( uma pista como “Pedrante” não é permitida se a palavra a ser adivinhada for Pedra), Uma palavra foneticamente idêntica à palavra Secreta, mas, cujo significado seja diferente ( a pista Sessão não é permitida se a palavra a ser adivinhada for Seção).
  • 3) Quando todos os jogadores tiverem escrito suas pistas, o jogador ativo deve fechar os olhos. Neste momento, os outros jogadores comparam as pistas escritas. Todas as pistas idênticas ou inválidas devem ser anuladas. Para anular uma pista, simplesmente remova-a da rodada.
  • 3.1) Pistas idênticas: Duas palavras idênticas (Rato e Rato), Variantes da mesma família (Príncipe e Princesa), Variantes da mesma palavra (Ator e Atriz, Casa e Casas): plurais, diferenciações de gênero e erros ortográficos não contam como diferenças.
  • 4) Após as pistas idênticas ou inválidas serem anuladas, peça para o jogador ativo abrir os olhos e tentar adivinhar a palavra Secreta com a ajuda das pistas remanescentes. Ele pode dar SOMENTE UM PALPITE.
  • 4.1) Sucesso: Se o jogador ativo adivinhar corretamente a palavra Secreta: o time ganha um ponto e avança para a próxima palavra.
  • 4.2) Fracasso: Se o jogador ativo der um palpite errado: não ganha ponto e perde a próxima palavra (perde a atual e a próxima).
  • 4.3) Se o jogador ativo escolher não responder, ele passa a sua vez: perde apenas a palavra atual.

FIM DO JOGO

O jogo termina quando não houver mais cartas no baralho. Conta-se os pontos e verifica a pontuação exibida no dispositivo.

VARIANTE PARA 3 JOGADORES

Mesma coisa, mas na hora de escrever pistas, cada jogador escreve duas pistas.
